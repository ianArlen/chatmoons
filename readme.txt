Se conecta a una base de datos de mongo atlas, por lo cual
np es necesario instalar nada de la base de datos. 
Solo se tiene que descargar y se debe correr primero el back,
porque lo deje en el puerto 3000 y el front si cambia el puerto. 

El back solo se corre con : 
npm run-script start-dev

El front se corre con : 
npm start

Despues de correrlo te puedes registrar en: 
http://localhost:3001/register

Despues de registrarte te puedes logear:
http://localhost:3001/login

Una vez logeado entra en: 
http://localhost:3001/

Entraras a varias opciones de chat, se puede iniciar
la conversacion con otra persona, repite el proceso de 
registrarte, logearte y entrar a http://localhost:3001/
y entra en la misma sala de chat, solo funciona con 
otra persona.
